# Document Management System

A Flask and MongoDB document management system for a university project.

## Features

- Create documents
- View all documents
- Update documents
- Delete documents
- Search documents
- Sort documents
- Pagination
- MongoDB Atlas integration

## Technologies Used

- Python
- Flask
- MongoDB Atlas
- PyMongo
- HTML
- CSS
